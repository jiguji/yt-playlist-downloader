## üèÅ Troubleshooting

If you encounter issues, here are some tips:

- **Script Not Running**: Make sure you used the `chmod` command to make the script executable.
- **yt-dlp Not Found**: Ensure that `yt-dlp` is installed correctly. Run `yt-dlp --version` to verify.
- **Network Issues**: Check your internet connection if the script cannot download videos.
